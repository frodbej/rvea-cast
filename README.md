## Overview

This project introduced **RVEA-CAST (Reference Vector-guided Evolutionary Algorithm for Cluster Analysis of Single-cell Transcriptomes)** to address the clustering of scRNA-seq data as a multi-objective optimization problem. Our approach considered three objectives to optimize: clustering deviation, clustering compactness, and the Davies–Bouldin index.
